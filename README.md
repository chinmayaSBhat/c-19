Some of the features were:
- Bed availability listing segregated by types in a given hospital.
- An admin portal for hospitals to update patient admission and discharge, which automatically obtains vacant and occupied bed counts, in REAL TIME!
- Automatically sort hospitals basis of user's location.
- SOS button to call nearest hospital basis user's location
- Filters on distance, bed availability ratio and user experience

Ambulance:
- Near realtime ( 5mins ) ambulance tracking.
- Ambulance occupancy/vacancy details
- Off duty/On duty toggle for drivers
- Sorting based on nearest ambulance
- Ambulance Driver/ Owner facing Android app for updating details realtime

Other facilities such as Labs, Pharmacists:
- Crowd updation
- Location based sorting
